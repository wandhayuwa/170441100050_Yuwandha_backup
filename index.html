    K-Nearest Neighbor¶

    DEFINISI¶

    Apa itu K-Nearest Neighbor

    Ø K-nearest neighbors atau knn adalah algoritma yang berfungsi untuk melakukan klasifikasi suatu data berdasarkan data pembelajaran (train data sets), yang diambil dari k tetangga terdekatnya (nearest neighbors). Dengan k merupakan banyaknya tetangga terdekat.

    Ø KNN digunakan dalam banyak aplikasi data mining, statistical pattern recognition, image processing, dll.

    Ø Tujuan dari algoritma ini adalah untuk mengklasifikasikan obyek baru berdasarkan atribut dan sampel2 dari data training.

    Ø Algoritma K Nearest neighbor menggunakan neighborhood classification sebagai nilai prediksi dari nilai instance(permisalan) yang baru.

    Klasifikasi Terdekat (Nearest Neighbor Classification)

    ​ Teknik pencarian tetangga terdekat yang umum dilakukan dengan menggunakan formula jarak euclidean*.* Berikut beberapa formula yang digunakan dalam algoritma knn.

    1. Euclidean Distance

    Teknik paling banyak digunakan. Jarak Euclidean adalah formula untuk mencari jarak antara 2 titik dalam ruang dua dimensi.

    2. Hamming Distance¶

    Jarak Hamming adalah cara mencari jarak antar 2 titik yang dihitung dengan panjang vektor biner yang dibentuk oleh dua titik tersebut dalam block kode biner.

    3. Manhattan Distance¶

    Manhattan Distance atau Taxicab Geometri adalah formula untuk mencari jarak d antar 2 vektor p,q pada ruang dimensi n*.*

    4. Minkowski Distance¶

    Minkowski distance adalah formula pengukuran antar 2 titik pada ruang vektor normal yang merupakan hibridisasi yang mengeneralisasi euclidean distance dan mahattan distance.

    Algoritma¶

    Tahapan Langkah Algoritma K-NN**¶

    1. Menentukan parameter k (jumlah tetangga paling dekat).
    2. Menghitung kuadrat jarak eucliden objek terhadap data training yang diberikan.
    3. Mengurutkan hasil no 2 secara ascending (berurutan dari nilai tinggi ke rendah).
    4. Mengumpulkan kategori Y (Klasifikasi nearest neighbor berdasarkan nilai k).
    5. Dengan menggunakan kategori nearest neighbor yang paling mayoritas maka dapat dipredisikan kategori objek.
    Banyaknya k Tetangga Terdekat¶

    ​ Untuk menggunakan algoritma k nearest neighbors, perlu ditentukan banyaknya k tetangga terdekat yang digunakan untuk melakukan klasifikasi data baru. Banyaknya k, sebaiknya merupakan angka ganjil, misalnya k = 1, 2, 3, dan seterusnya. Penentuan nilai k dipertimbangkan berdasarkan banyaknya data yang ada dan ukuran dimensi yang dibentuk oleh data.

    IMPLEMENTASI¶

    UC Irvine Machine Learning Repository

    ​ Data yang digunakan dibawah ini adalah Iris Flower Dataset(IFD) yang pertama kali diperkenalkan pada tahun 1936 oleh ahli statistic terkenal Ronald Fisher dan terdiri dari 50 pengamatan dari masing- masing 3 spesies Iris (Iris Setosa, Iris Virginica, dan Iris Versicolor). Empat fitur diukur dari masing-masing sampel : sepal_length, sepal_width, petal_lenght, petal_width.

    Script dibawah ini menggunakan python.

    #memuat/membuat library
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
import matplotlib.pyplot as plt
import pandas as pd
from sklearn.neighbors import KNeighborsClassifier
import numpy as np

#dibawah ini memuat/memamnggil data dengan load_iris dari sklearn
iris_dataset = load_iris()

# menampilkan informasi tentang data iris
print("Keys of iris_dataset \n{}".format(iris_dataset.keys()))
print(iris_dataset['DESCR'][:193] + "\n...")
print(" Nama Kelas: {}".format(iris_dataset['target_names']))
print("Nama Fitur: {}".format(iris_dataset['feature_names']))
print("Type of data: {}".format(type(iris_dataset['data'])))
print("Shape of data: {}".format(iris_dataset['data'].shape))

# menampilkan 10 data iris untuk digunakan sebagai data training
print("Data Training: \n{}".format(iris_dataset['data'][:10]))

# Type of target
print("Type of target: {}".format(type(iris_dataset['target'])))
print("Shape of target: {}".format(iris_dataset['target'].shape))

# untuk hasil tampilannya nanti : 0 = Setosa, 1 = Versicolor, 2 = Virginica
print("Target: \n{}".format(iris_dataset['target']))

# Training and Testing Data
X_train, X_test, y_train, y_test = train_test_split(iris_dataset['data'],
                                                    iris_dataset['target'],
                                                    random_state=0)
print("X_train shape: {}".format(X_train.shape))
print("y_train shape: {}".format(y_train.shape))
print("X_test shape: {}".format(X_test.shape))
print("y_test shape: {}".format(y_test.shape))

# membuat dataframe dari data X_train
#memberi label pada kolom menggunakan string di iris_dataset.feature_names
iris_dataframe = pd.DataFrame(X_train, columns=iris_dataset.feature_names)

# membuat matriks pencar dari bingkai data, mewarnai y_train
g = pd.scatter_matrix(iris_dataframe, c=y_train, figsize=(15, 15), marker='o',
                      hist_kwds={'bins': 20}, s=60, alpha=.8)
plt.show()

# membuat model menggunakan KNN
knn = KNeighborsClassifier(n_neighbors=1)
knn.fit(X_train, y_train)

X_new = np.array([[5, 2.9, 1, 0.2]])
print("X_new.shape: {}".format(X_new.shape))

prediction = knn.predict(X_new)
print("Prediksi: {}".format(prediction))
print("Nama target yang diprediksi: {}".format(iris_dataset['target_names'][prediction]))

# Evaluating model
y_pred = knn.predict(X_test)
print("kumpulan tes prediksi: \n {}".format(y_pred))
print("hasil set tes: {:.2f}".format(np.mean(y_pred == y_test)))
print("hasil set tes: {:.2f}".format(knn.score(X_test, y_test)))
